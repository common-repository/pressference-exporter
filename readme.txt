Export WooCommerce store data including Orders, Products, Customers, Reviews, Categories, Attributes, Tag and Coupon to various output format.

== Description ==

PF Exporter plugins support the following WooCommerce data types Products, Categories, Tags, Orders, Customers, Coupons, Reviews and Attributes. PF Exporter plugin simplify the process of export data, by allowing user to filter selected data and creating profile to automate export process.

Purchase pro version of [**PF Exporter Pro**](https://pressference.com) to unlock more features

Supported export destination:
* Direct Download
* Local Server
* Dropbox
* Google Drive
* Email

Supported file type are:
* CSV
* Excel
* TSV
* jSON
* XML

Additional feature: 
* Export data filter
* Automate export feature
* Export field mapping
* Export field ordering sequence
* Export field customize
* Bulk export profile
* More feature to come... :)

== Installation ==

By browsing in wordpress marketplace:

1. go to Plugins, click Add New
2. search for PF Exporter
3. click “Install Now”
4. activate and navigate to PF Exporter admin menu.


By manually uploading plugins:

1. download the plugin (.zip file).
2. upload to /wp-content/plugins/ and unzip.
3. activate plugin on Plugins page.
